# Synopsis

Unbreakable is an easy difficulty challenge that features abusing python `eval()`, bypassing a blacklist of characters and manage to read flag.

# Description

Think you can escape my grasp? Challenge accepted! I dare you to try and break free, but beware, it won't be easy. I'm ready for whatever tricks you have up your sleeve!

## Skills Required

- Basic python.

## Skills Learned

- Using `eval()` to call `print` and other functions to read `flag.txt`.

The challenge is pretty straightforward. Our input is `stripped` and then the `any` function searches for the `blacklist` characters and strings inside the given input. Many characters and strings are banned, e.g. `import`, `;` etc, thus limiting the potential of our payload. Then, user input is passed to `eval()` along with `()`. The goal always is to `get shell` or `read flag.txt`. This time, will go for the latter. 

### Exploitation path

As we can see, we can call all functions that do not contain the letters `s` and `b`. Some good candidates are: 

* `print` to write to `stdout`
* `open` to open the `flag.txt`
* `read` to read its content
